# 🏋️‍♂️ Iron & Forge – Modern Gym Landing Page

A premium, high-performance landing page designed for a hardcore & modern fitness center named **Iron & Forge**. This project demonstrates a component-driven frontend architecture focused on visual impact, clean code organization, and a seamless user experience.

🔗 **Live Demo:** [https://diachukmatviy.github.io/Iron---Forge/](https://diachukmatviy.github.io/Iron---Forge/)

✨ Key Features
Component-Driven Architecture: Built with React 18 for high reusability and maintainable UI structure.

Scoped Component Styling: Powered by CSS Modules (*.module.css) to eliminate global style conflicts and ensure strict component encapsulation.

Smooth Reveal Animations: Integrated a custom React wrapper (<FadeIn />) to handle performance-friendly scroll animations.

Fully Responsive Design: Perfectly optimized for all screen sizes, from mobile devices to desktop monitors.

🧩 Core Sections Included:
Hero: Impactful intro screen with high-converting Call-to-Action (CTA) elements.

Benefits: A structured breakdown of what makes the gym unique.

Trainers: Profile showcase of the coaching staff.

Transformation: Interactive Before/After visual section to capture user attention.

Pricing: Clean, transparent tables for membership plans.

Testimonials & FAQ: Social proof slider/grid paired with functional accordion drop-downs.

🛠️ Tech Stack & Architecture
Core Framework: React 18

Build Toolchain: Vite (for ultra-fast bundling and optimized performance)

Styling Method: Vanilla CSS + CSS Modules

State & Helpers: Custom React hooks (/hooks) and helper configurations (/utils)

📂 Directory Overview (As seen in image_a17c4b.png):
src/components/ — Isolated UI components, each bundled with its own logic (.jsx) and private styles (.module.css).

src/hooks/ — Custom hooks abstracting repetitive functional logic.

src/utils/ — Helper files and static structural data.

src/App.jsx & main.jsx — Core layout orchestration and DOM entry point.
